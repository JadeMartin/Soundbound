# SENG302 Project Team 700
This is Travel EA, a website for storing and organising destinations and trips.
It can be packaged to a deliverable form using the `sbt dist` command.

## How to run
Start the Play app:
```bash
sbt run
```
And open <http://localhost:9000/>
